🧠 Output hierarchy

The report should be interpreted in layers:
1. OCR text 2. Layout / sections 3. Medical entities 4. Biomarker meaning 5. Safety filters 6. Suggested next steps
Example: glucose 260 → high blood sugar / diabetes risk; haemoglobin 8 → anemia risk; blood pressure 160/100 → hypertension concern.
Safety-first medicines: age, weight, allergies, pregnancy, chronic disease, and report context must be checked before any suggestion is shown.
Important: Handwritten prescriptions in India should not be auto-turned into a fake prescription. The app should only extract and explain, then suggest doctor review.
